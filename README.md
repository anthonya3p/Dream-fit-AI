# DreamFit AI — GitHub / Vercel

Application PWA mobile premium pour iPhone.

## Installation sur GitHub
1. Crée ou ouvre le dépôt GitHub utilisé par l'application.
2. Supprime les anciens fichiers du dépôt.
3. Envoie tout le contenu de ce dossier à la racine du dépôt.
4. Vérifie que `index.html` se trouve bien à la racine.
5. Dans Vercel, relance un déploiement si nécessaire.

## Installation sur iPhone
1. Ouvre l'adresse Vercel dans Safari.
2. Touche Partager.
3. Choisis « Sur l'écran d'accueil ».
4. Supprime l'ancienne icône puis réinstalle l'application si iOS conserve l'ancien cache.

## Données
Les données sont stockées localement sur l'appareil avec `localStorage`.
Le menu « Plus » permet d'exporter un fichier JSON de sauvegarde.
